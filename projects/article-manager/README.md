# 文章管理系统

本地网页版文章管理测试系统，Apple 风格界面。

## 快速开始

```bash
npm install
node server.js
```

浏览器打开 http://localhost:3000

- 账号：admin
- 密码：123456

## 功能

- 登录验证
- 文章卡片展示（缩略图 + 标题 + 作者 + 预览）
- 全文阅读
- 文章上传（含配图）
- 文章编辑
- 文章删除
- 操作日志
- 数据统计

## 技术栈

- 后端：Node.js + Express
- 前端：纯 HTML/CSS/JS
- 存储：本地 JSON 文件 + 文件系统
